# BTS Quiz App (MERN Full Stack)

A **full-stack quiz application** built using **MongoDB, Express.js, React.js, and Node.js (MERN)**.  
This app allows users to take quizzes, track scores, and store results in a database. It includes a **client (frontend)** and **server (backend)** folder.

---

## Project Overview

- Users can register and log in  
- Take BTS-themed quizzes  
- View quiz results and track scores  
- Admin (optional) can manage quizzes and questions  

---

## Tech Stack

- **Frontend (Client):** React.js, Redux, CSS, HTML  
- **Backend (Server):** Node.js, Express.js  
- **Database:** MongoDB Atlas  
- **Authentication:** JWT (JSON Web Token)  
- **Other:** Redux for state management  

---

## Features

- User signup and login  
- Quiz selection and timer  
- Dynamic quiz questions with multiple-choice answers  
- Score calculation and result tracking  
- Admin panel to manage quizzes (optional)  
- Responsive design for desktop and mobile
